## Функции

Приложение получает данные о самолетах, находящихся сейчас в воздухе, с помощью открытого API https://opensky-network.org/ — и загружает полученные данные в созданные таблицы (в БД PostgreSQL).



## Установка:



pip install requests

poetry add requests

pip install -r requirements.txt



Запуск



python main.py
